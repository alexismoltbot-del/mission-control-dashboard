**Post 4/6**

Système de crédits :

Problème initial : coûts explosent si enrichissement illimité.

Solution :
- Recherches illimitées (coût quasi-nul)
- Enrichissement = crédits
- 1 crédit = 1 email (0,15€)
- 10 crédits = 1 téléphone (1,50€)

Users paient uniquement ce qu'ils consomment.

Win-win.

---

**Post 5/6**

Code sample (recherche Exa) :

```javascript
const results = await exa.search({
  query: userPrompt,
  numResults: 25,
  useAutoprompt: true,
  type: "neural"
});

const enriched = await Promise.all(
  results.map(async (company) => {
    const siren = await pappers.getBySiren(company.id);
    const email = await dropcontact.findEmail(siren.dirigeant);
    return { ...company, siren, email };
  })
);
```

Simple. Efficace.
