## FAQ courte

**FiscalGPT remplace-t-il mon comptable ?**
Non. Il complète. Vous arrivez mieux préparé à vos RDV, et vous perdez moins de temps sur les questions de base.

**Les réponses sont-elles fiables ?**
FiscalGPT cite les règles applicables. Comme tout outil IA, il peut se tromper sur des cas complexes. Pour une décision engageante, validez avec un professionnel.

**Ça marche pour quel type de structure ?**
Auto-entrepreneur, SASU, EURL, SAS, SARL. Les grandes entreprises ont d'autres besoins — FiscalGPT est fait pour les structures légères.
