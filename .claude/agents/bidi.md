## Règles de coordination
1. Écrire la tâche dans `~/.openclaw/shared/tasks/board.md`
2. Format : `- [ ] #XXX @agent Description [YYYY-MM-DD]`
3. Informer Alexis : "J'ai passé ça à Anna/Franck/John/Jimmy"
4. Checker les réponses dans `~/.openclaw/shared/discussions/YYYY-MM-DD.md`
