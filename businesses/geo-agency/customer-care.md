## 3. Checklist onboarding — ce que l'équipe vérifie à chaque nouveau client

À réaliser dans les 24h suivant l'achat :

- [ ] Commande reçue et montant vérifié dans Stripe
- [ ] Email 1 (confirmation d'achat) envoyé automatiquement ou manuellement
- [ ] Compte client créé dans l'espace client
- [ ] Email 2 (accès compte) envoyé avec le bon lien
- [ ] Email 3 (intake) envoyé — lien formulaire fonctionnel testé avant envoi
- [ ] Intake reçu et relu (complet ? lisible ? informations suffisantes ?)
- [ ] Email 4 (confirmation intake + délai) envoyé avec le délai réel estimé
- [ ] Dossier client créé dans le tracker interne (nom, URL, date achat, délai)
- [ ] Première URL ou page prioritaire identifiée dans le formulaire
- [ ] Aucune anomalie côté paiement (montant, devise, nom client)

## 4. FAQ client — 5 questions avec réponses prêtes à envoyer

### Q1 — "Quand est-ce que vous commencez ?"
```
Dès réception de votre formulaire d'intake, nous démarrons. En général,
l'audit est lancé dans les 24h ouvrées suivant votre réponse.
```

### Q2 — "Combien de temps pour avoir mon livrable ?"
```
Le délai standard est de [X jours ouvrés] après réception de l'intake.
Ce délai vous est confirmé par email dès qu'on a reçu vos informations.
```

### Q3 — "C'est quoi exactement le livrable ?"
```
Vous recevez un audit GEO structuré qui identifie les points faibles de
votre site pour la visibilité dans les moteurs IA (ChatGPT, Perplexity,
Google SGE…). Il inclut les corrections prioritaires et les actions à mener,
organisées par impact.
```

### Q4 — "Est-ce que vous implémentez les changements vous-mêmes ?"
```
L'offre de base inclut l'audit et les recommandations. L'implémentation
est disponible en option — écrivez-nous si vous voulez un devis pour qu'on
s'en occupe directement.
```

### Q5 — "Je n'ai pas reçu mes accès / l'email est tombé dans les spams"
```
Désolé pour ça. Regardez dans vos spams / promotions avec l'expéditeur
"GEO Agency". Si vous ne trouvez pas, répondez à cet email ou écrivez à
support@geoagency.fr — on vous renvoie le lien dans l'heure.
```

## 5. Escalade à Alexis — critères et protocole

### Escalader immédiatement à Alexis si :

- Le client exprime une insatisfaction forte ou menace un remboursement
- Une promesse de délai n'a pas été tenue (livrable en retard de plus de 2 jours)
- Le client pose une question commerciale hors périmètre (partenariat, offre sur-mesure, volume)
- Une anomalie de paiement est détectée (doublon, montant incorrect)
- Le client demande à parler directement à "la direction" ou "au fondateur"
- La situation sort du scope décrit dans la FAQ et ne peut pas être résolue en 1 email

### Comment escalader :

1. Répondre au client : _"Je transmets votre message directement à Alexis,
   qui vous répondra personnellement sous [délai]."_
2. Envoyer à Alexis : contexte (1 ligne), historique des échanges, action attendue
3. Délai à annoncer au client : **1 jour ouvré maximum**

### Ce que Emma ne tranche pas seule :

- Remboursements
- Modifications d'offre ou gestes commerciaux
- Engagements sur des délais supplémentaires hors contrat
- Toute réponse qui engage la réputation de GEO Agency
