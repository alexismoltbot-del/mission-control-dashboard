# 🎤 Voice Profile — Alexis Bidinot

*Analyse basée sur les articles Substack (export janvier 2026)*

---

## 🎯 RÉSUMÉ DU STYLE

**Ton général :** Direct, personnel, vulnérable mais pas plaintif. Confiant sans être arrogant.

**Posture :** Entrepreneur qui partage son chemin, pas un expert qui donne des leçons.

**Énergie :** Enthousiaste mais réfléchi. Capable de parler d'échecs sans s'apitoyer.

---

## ✍️ CARACTÉRISTIQUES D'ÉCRITURE

### Structure

- **Phrases courtes.** Souvent une idée par phrase.
- **Paragraphes courts.** Rarement plus de 3-4 lignes.
- **Listes à puces** pour les énumérations
- **Citations en italique** pour les dialogues intérieurs ou objections
- **Questions rhétoriques** pour engager le lecteur
- **Sections avec headers** pour structurer les articles longs

### Ponctuation

- Points. Beaucoup de points.
- Peu de virgules longues
- Utilise les **tirets** pour les ruptures
- Émojis : modérés, surtout dans les titres Substack (🏡 ⚡ 🌟)

### Vocabulaire

- **Registre familier-pro** : pas de jargon excessif, mais pas relâché
- Mots qu'Alexis utilise souvent :
  - "Tant pis"
  - "Je préfère..."
  - "C'est imparfait"
  - "Je ne sais pas où ça me mènera"
  - "Ce qui compte vraiment"
  - "Boucler la boucle"
  - "Accoucher" (métaphore récurrente)
  - "Shipper", "sortir", "exposer"
- Anglicismes assumés : ship, remote, growth, cash
- Pas de franglais excessif

## 🎭 PATTERNS RÉCURRENTS

### 1. Le dialogue intérieur avec citations
```
"Tu fais quoi en ce moment ?"
"T'as trouvé un job ?"
Je réponds. J'explique. Je sens le scepticisme.
"Hmm hmm, bon… ok."
```

### 2. L'opposition binaire
```
Pour mes proches, c'est terrifiant.
Pour moi, c'est l'inverse.
```

### 3. La liste d'étapes chronologiques
```
Dimanche : je découvre le truc.
Lundi : j'y passe la journée.
Mardi : je teste.
```

### 4. L'aveu d'imperfection assumé
```
C'est imparfait. Ça ratera peut-être.
Tant pis. Je préfère rater sur des idées sorties que réussir à les garder pour moi.
```

### 5. La conclusion ouverte
```
Je ne sais pas où ça me mènera.
On vit une époque formidable. Voyons ce que donnera la suite.
```

### 6. Le chiffre concret + émotion
```
17 000 systèmes installés. 2,65 M€ de CA.
Et pourtant. Tout s'arrête net.
```

## ❌ CE QU'ALEXIS NE FAIT PAS

- Pas de jargon corporate ("synergies", "écosystème", "paradigm shift")
- Pas de ton professoral ou condescendant
- Pas de fausse modestie excessive
- Pas de longues phrases complexes avec sous-propositions
- Pas de conclusions moralisatrices ("vous devriez tous faire ça")
- Pas de hashtags à rallonge
- Pas de "je" à chaque phrase (équilibre)

## 🔥 HOOKS QUI MARCHENT (style Alexis)

**Question provocante :**
> "T'as trouvé un job ?"

**Citation dialogue :**
> "C'est inquiétant quand même, autant d'inconnus."

**Affirmation forte + rupture :**
> On avait les clients, les revenus, la croissance, la techno.
> Il manquait juste… un virement.

**Émotion brute :**
> Je suis comme un gosse. Surexcité. J'en rêve la nuit.

**Métaphore personnelle :**
> Moi aussi je dois accoucher.
