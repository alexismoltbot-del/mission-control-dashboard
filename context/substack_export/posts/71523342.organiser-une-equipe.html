Netflix org culture

Quelques rituels nous aident bien

  • Quotidien :

    • Les daily (standup) pour toute l’équipe tech/produit.

    • + 1 daily focus app pour accélérer (👥 : Product + design + Dev d’app)

  • Hebdomadaire :

    • Point management. Tour de tous les sujets : produit, dev, infras, production (usines), marketing, RH, …

    • Démo produit

  • Mensuel : All-hands.

Feature Team

En fonction de vos produits, l’organisation par “feature team” semble bien fonctionné. Elle a fait ses preuves. Il faut être à minima une 20aine de tech/dev pour que les postes soient doublés.

Lorsque c’est possible Elle consiste souvent en :

  • 1 x product

  • 1 x designer

  • 1x back-end

  • 1 x dev iOS

  • 1x dev Andoïd

  • + développeur embarqué.

Début septembre 2022, nous sommes 18, dont 14 dans l’équipe tech/dev. Nous ne sommes pas assez nombreux pour structurer. Nous avons un produit à sortir, tout le monde tire dans le même sens.
